#### 搜开发者

| 搜索条件    | 备注                                           |
| ----------- | ---------------------------------------------- |
| followers:  | followers:>=1000，匹配超过1000名关注者的开发者 |
| language:   | language:java,匹配开发语言为 java 的开发者     |
| in:fullname | jack in:fullname,匹配用户实名为 jack 的开发者  |
| location:   | location:china,匹配用户填写的地市在 china      |

#### 搜项目及工具

| 搜索条件               | 备注                                                         |
| ---------------------- | ------------------------------------------------------------ |
| Awesome + keyword      | 关键字 Awesome，帮助找到优秀的工具列表                       |
| in:name example        | in:name springboot ,匹配项目名中包含 springboot 关键词的项目 |
| in:readme example      | in:readme sprinboot,匹配 readme.md 中包含 spingboot 关键词的项目 |
| in:description example | in:description springboot,匹配描述中包含 springboot 关键词的项目 |
| stars:                 | stars:>=1000,匹配收藏数量等于大于 1000 的项目                |
| forks:                 | forks:>=1000,匹配分组数量等于大于 1000 的项目                |
| pushed:                | pushed:>2021-08-25,匹配在2021年8月25号后有更新的项目         |
| language:              | language:java,匹配以 java 开发的项目                         |
| size:                  | soze:>=4000,匹配开源项目包小大于等于4000K的项目              |
| license:               | 开源协议                                                     |
| user:                  | 作者                                                         |
| create:                | create:>=2021-08-25,匹配2021年8月25后创建的项目              |
